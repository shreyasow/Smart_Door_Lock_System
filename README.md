# Smart Door Lock System with Fingerprint Authentication

## Overview
IoT-based smart door lock system using ESP32 and fingerprint sensor with cloud logging and real-time access control.

## Features
- Fingerprint authentication (enroll, verify, delete)
- Servo-based door locking mechanism
- Wi-Fi enabled IoT system using ESP32
- Cloud logging using Google Sheets API
- Intruder detection with image capture
- LCD and audio feedback system

## Hardware Used
- ESP32
- R305/R307 Fingerprint Sensor
- SG90 Servo Motor
- DF Player Mini + Speaker
- 16x2 I2C LCD

## Software Used
- Arduino IDE (C++)
- Python
- Google Sheets API
